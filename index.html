2D Tower Defense

Player Controls

  • Move Left: Arrow Left
  • Move Right: Arrow Right
  • Move Up: Arrow Up
  • Move Down: Arrow Down
  • Place Tower: T
  • Place Fence: F
  • Toggle Debug Mode: D

Game Description

Welcome to 2D Tower Defense! Your objective is to defend your base from waves of enemies by strategically placing towers and fences on the battlefield.

Use the arrow keys to move your character and the 'T' key to place towers. Fences can be placed using the 'F' key. Monitor the debug mode by pressing 'D' to see the debug.
